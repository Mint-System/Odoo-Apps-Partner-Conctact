================================
Partner Contact Location in Name
================================

Show zip and city in partner name.

For a detailed documentation have a look at https://www.odoo-wiki.org/.

Configuration
~~~~~~~~~~~~~

* Set context ``'show_zip_and_city': True`` on ``res.partner`` fields.
